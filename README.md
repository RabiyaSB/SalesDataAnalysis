# 📊 AI-Powered Sales Dashboard (Power BI)

## 🚀 Project Overview
This project presents an interactive Power BI dashboard designed to analyze sales performance, product trends, and regional insights.

## 🔧 Tools Used
- Power BI
- DAX
- Data Modeling
- Time Intelligence (YoY, Forecasting)

## 📈 Key Features
- Revenue, Profit, and KPI tracking
- Year-over-Year (YoY) growth analysis
- Sales forecasting and trend analysis
- Product and regional performance insights
- AI visuals (Smart Narrative, Decomposition Tree)
